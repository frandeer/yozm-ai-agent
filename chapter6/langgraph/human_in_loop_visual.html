3 기억해야 할 3가지

🛑

1. 멈춘다

AI가 작업 중간에 잠시 멈춥니다. 마치 신호등의 빨간불처럼!

input("답변: ")

← 여기서 멈춤!

👤

2. 사람이 개입

사람이 정보를 제공하거나 결정합니다. 사람이 중요한 역할!

예시:

✅ 추가 정보 제공

✅ 승인/거부 결정

🔄

3. 계속한다

사람의 답을 받으면 AI가 다시 작업을 계속합니다. 루프(반복)가 가능!

가능한 흐름:

AI → 사람 → AI → 완료

AI → 사람 → AI → 사람 → AI → 완료

# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# 1️⃣ AI 노드: 질문 생성
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
def ai_node(state):
    if state.has_details:
        # 정보 충분 → 작업 완료
        return "작업 완료!"
    else:
        # 정보 부족 → 질문
        return "어떤 보고서인가요?"

# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# 2️⃣ 사람 노드: 입력 받기 ⭐
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
def human_node(state):
    # 🛑 여기서 멈춤!
    answer = input("답변: ")
    return answer

# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# 3️⃣ 그래프: 노드 연결
